This repo was pushed using Git Bash through SSH Key

This repository was uploaded to GitHub using Git command, and also the HTML and CSS files was edited using only nano editor though Git Bash (You may see the code not organized as it could using VS Code)

I will push the repository befor and after some modifications

The Commands that I used

Commands to start and prepare the Repo

Command What do Note
git init Start and create your Git repo This is the firt and must applied command
git add . (or) git add TheFileName.itsExtension Add files and folders to your local Git repo to be commit later The "." after the "Git add" defin the whole files and folder, but if we mentioned a file or folder in specific instead of typing "." it will add the specific file or folder.
git status To know the status of our Git repo Sometimes its usful to know what the next step to do
git commit -m "some message" to record a written message about the changes some times we can use "git commit -am" instead of git commit -m

Command to create SSH key and upload the repo to github

  1. git remote add origin url-of-the-github-repo
  2. eval $(ssh-agent -s)
  3. ssh-keygen
  4. ssh-add the-name-and-source-of-the-ssh-key-file
  5. git push -u origin master
